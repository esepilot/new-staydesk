# Changelog

All notable changes to StayDesk will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of StayDesk by BendlessTech
- Hotel authentication system with email confirmation
- Hotel dashboard with real-time metrics
- Booking management system
- Room management functionality
- Paystack payment integration for Nigerian Naira (NGN)
- Subscription system (Monthly: ₦49,900, Yearly: ₦598,800)
- 10% discount for first 10 hotels (yearly plan only)
- AI chatbot with bilingual support (English & Nigerian Pidgin)
- Email notifications system
- WhatsApp integration for guest notifications
- Admin dashboard for platform management
- RESTful API endpoints for external integrations
- Apple-inspired design with smooth animations
- Responsive design for all devices
- WhatsApp widget fixed in lower right corner
- Custom WordPress roles and capabilities
- Database tables for hotels, rooms, bookings, guests, transactions, chat logs, subscriptions, and support tickets
- Settings page for Paystack, email, and API configuration
- Uninstall cleanup functionality

### Features
- Comprehensive booking workflow
- Payment verification and refund management
- Guest enquiry handling through chatbot
- Subscription expiry checking with cron jobs
- Dashboard alerts and notifications
- Real-time clock display
- Number counting animations
- Staggered card reveal animations
- Toast notifications
- Modal dialogs
- Form validation
- AJAX-powered interactions

### Security
- WordPress coding standards compliance
- Proper data sanitization and validation
- Secure password handling
- Nonce verification for AJAX requests
- Role-based access control
